## Installation

Clone the repo
```
git clone git@github.com:valor-dev1/laravel-rest-api-blog.git
```

If you want to run on docker

```
docker compose up -d
```

Then run the following commands (in the container for Docker)

```
composer install
cp .env.example .env // Update  database credentials
php artisan migrate
```
